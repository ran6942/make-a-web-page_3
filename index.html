Lesson 1: The Basics of the Web & HTML

Vocabulary

  • World Wide Web: A collection of html documents
  • Internet: World's largest computer network
  • Browser: A program that runs on your computer to display files found on the web
  • Http: Main protocol of the web
  • Server: A computer that's optimized for hosting files that make up the web
  • HTML: Hyper Text Markup Language. HTML is the main document of the web. It's the basis for everything on the web.
  • Tag: Contained with angled brackets. Non-void tags have an opening & closing tag
  • Elements: Refers to everything set within opening & closing tag
  • Attribute: Property of html element

How the Web Works

The main components of the web are:

  1. 1) The User + Computer & Browser
  2. 2) The Internet
  3. 3) Servers
  4. 4) Files

The browser that the user is running on his computer makes requests using the http protocol via the internet to the server. When the server receives the request, it responds by returning files to the browser. The browser then interprets and displays the files.

HTML (Hyper Text Markup Language)

HTML is the main document of the web. HTML is the main structure of web pages.

HTML is made up of:

  1. Text content - what you see
  2. Markup - what it looks like
  3. References to other document
  4. Links to other pages

HTML markup is made of elements, which are made of tags.

Elements can be block or inline elements. Block element content is contained within an invisible box, whereas inline element content is contained within their tags and does not start a new line.

Main Takeaways

  1. A webpage is a text document written in html. Web browsers interpret and render these documents. These documents are stored on a server.
  2. Coding is writing text in a language that the computer can understand. The computer reads these instructions and executes the instructions.
  3. Computers are stupid. Programmers must write exactly the way a computer understands. The syntax of the language must be precise and accurate. Computers don't have the capability to understand context
  4. Programmers can't remember everything, so don't stress out about memorizing everything.

Lesson 2: Creating a Structured Document with HTML

Vocabulary

  • CSS (Cascading Stylesheets): CSS is a language used to control the styling components of a web page.
  • DOM (Document Object Model): The DOM is a cross-platform, language agnostic convention for representing and interacting with objects in html.
  • Javascript: Javascript controls the interactive components of a webpage.

Document Object Model (DOM)

The browser builds a tree-like, branching structure of a web page, otherwise known as the DOM tree. The browser turns html tags into elements that form nodes of the tree. The DOM is an alternative representation of an HTML document.

Mozilla Developer Network provides a comprehensive introduction to the DOM HERE.

Boxes, Boxes, Boxes

Everything on a web page is a box. You can break down web page elements into boxes nested within boxes, nested within boxes.

Using a browser's developer tools, one can access any web page source and observe this. Understanding this concept of elements being contained in a box can be very helpful when laying out contents of a web page.

CSS-Tricks has a easy to understand article about the box model.
